How to Play - Jurassic Adventure

Controls

  • SPACE - Jump
  • CLICK/TAP - Jump
  • Double Tap - Double jump for rivers
  • P key - Pause/Resume
  • R key - Restart game

Obstacles

  • Trees - Jump over them
  • Rocks - Jump over them
  • Rivers - Double jump required!
  • Other Dinosaurs - Jump over rivals

Game Rules

  • Start slow - Speed increases at higher levels
  • You have 3 lives - Hit obstacle = lose 1 life
  • Score increases over time
  • Every 100 points = New level
  • Higher levels = Faster gameplay

Pro Tips

  • Rivers need quick double jumps
  • Use Auto mode to learn timing
  • Pause when you need a break
  • Watch for patterns in obstacles
  • Don't panic - game starts slow!
